# Редизайн Admin панели — дизайн-документ

**Дата:** 2026-03-07

## Цель

Привести админку к уровню пользовательского dashboard — темная тема, анимации, сворачиваемый сайдбар, управление пользователями, расширенная аналитика.

## 1. Layout (layout-client.tsx)

- Темная тема — убрать AdminThemeForcer, поддержка dark mode
- Сворачиваемый сайдбар — как в ЛК: localStorage, тултипы, анимация collapse
- Хлебные крошки на всех подстраницах
- Анимации переходов между страницами (Framer Motion AnimatePresence)
- Профиль админа в нижней части сайдбара с dropdown

## 2. Навигация (новая структура сайдбара)

- Заявки (главная)
- Пользователи (новая страница)
- Аналитика
- Настройки
- Убрать страницу "Услуги"

## 3. Главная — Заявки (page.tsx)

- Статистика — виджеты в стиле dashboard (градиенты, иконки, тени, анимация при наведении)
- Темная тема для таблицы, канбана, модалок
- Анимации — появление карточек, переключение видов
- Функциональность остается как есть

## 4. Новая страница — Пользователи (/admin/users)

- Таблица пользователей — имя, email, телефон, компания, дата регистрации, статус (активен/забанен)
- Поиск и фильтры — по имени/email, по статусу
- Действия над пользователем:
  - Просмотр профиля (оборудование, заявки, организации)
  - Редактирование данных
  - Сброс пароля (генерация нового)
  - Бан/разбан
  - Удаление аккаунта
  - Вход от имени пользователя (impersonate)
- API: GET/PATCH/DELETE /api/admin/users, POST /api/admin/users/[id]/ban, POST /api/admin/users/[id]/impersonate
- Prisma: добавить поле `banned Boolean @default(false)` в модель User

## 5. Аналитика — расширение

- Визуальная полировка — градиенты, тени, анимации как в dashboard
- Новые метрики:
  - Рост регистраций (график по месяцам)
  - Активность пользователей (входы за период)
  - Топ-пользователи по количеству заявок/оборудования

## 6. Настройки (settings/page.tsx)

- Темная тема
- Анимация переключения табов
- Визуальная полировка форм

## Что НЕ меняется

- Socket.IO real-time в заявках
- Канбан-доска
- Экспорт CSV/Excel
- Логика статусов заявок
